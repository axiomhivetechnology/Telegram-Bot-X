# Credible References & Resources

A curated list of high-quality resources for Telegram Bot development.

## Official Resources
-   [Official Telegram Bot API Documentation](https://core.telegram.org/bots/api) - The definitive guide to every method and type.
-   [Telegram Bot Features](https://core.telegram.org/bots/features) - Overview of what bots can do.
-   [BotFather Commands](https://core.telegram.org/bots#6-botfather-commands) - How to configure your bot's profile.

## Python Ecosystem
-   [python-telegram-bot Documentation](https://docs.python-telegram-bot.org/) - Extremely popular and well-documented.
-   [aiogram Documentation](https://docs.aiogram.dev/) - Modern, asynchronous framework for advanced bots.
-   [Pyrogram](https://docs.pyrogram.org/) - A powerful MTProto library.

## Node.js Ecosystem
-   [Telegraf.js](https://telegraf.js.org/) - The most widely used Node.js framework.
-   [grammY](https://grammy.dev/) - A newer, high-performance framework with great TypeScript support.

## Community & Tutorials
-   [Telegram Bots Subreddit](https://www.reddit.com/r/TelegramBots/)
-   [Official Bot News Channel](https://t.me/BotNews)

## Tools
-   [Postman Collection for Bot API](https://github.com/TelegramBots/postman-telegram-bot-api) - Test API calls without code.
-   [ngrok](https://ngrok.com/) - Expose your local server to the internet (useful for testing webhooks).

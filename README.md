# Telegram-Bot-X

This project contains a Telegram AI Assistant Bot and a guide for exploring Telegram content.

## AI Assistant Bot
A powerful AI agent assistant that can perform tasks and answer questions directly on Telegram.

### Setup
1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file and add your `BOT_TOKEN` and `OPENAI_API_KEY`.
4. Run `node index.js`.

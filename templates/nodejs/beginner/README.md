# Beginner Node.js Bot

A simple "Echo Bot" built with the **Telegraf** framework.

## How to Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Set your Token**: Create a `.env` file and add:
    ```text
    TELEGRAM_TOKEN=your_token_here
    ```
3.  **Run the bot**:
    ```bash
    npm start
    ```

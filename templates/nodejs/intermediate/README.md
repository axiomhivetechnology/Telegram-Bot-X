# Intermediate Node.js Bot

This template uses **Telegraf Scenes** to manage multi-step user interactions.

## Features
- **Session Management**: Remembers data across multiple messages.
- **Wizard Scenes**: Simplifies linear flows (like filling out a form).
- **Validation**: Basic input validation example.

## How to Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Set your Token**: Create a `.env` file with `TELEGRAM_TOKEN`.
3.  **Run the bot**:
    ```bash
    npm start
    ```

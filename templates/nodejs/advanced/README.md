# Advanced Node.js Bot (grammY)

This template uses **grammY**, a modern and high-performance framework for Telegram bots.

## Features
- **Speed**: Optimized for performance and low memory usage.
- **Type Safety**: Excellent support for TypeScript (though this example is JS).
- **Scalability**: Designed for bots with millions of users.

## How to Run

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Set your Token**: Create a `.env` file with `TELEGRAM_TOKEN`.
3.  **Run the bot**:
    ```bash
    npm start
    ```

# Beginner Python Bot

This is a simple "Echo Bot" that greets you and repeats everything you say.

## How to Run

1.  **Install requirements**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Set your Token**: Create a `.env` file in this folder and add:
    ```text
    TELEGRAM_TOKEN=your_token_here
    ```
3.  **Run the bot**:
    ```bash
    python bot.py
    ```

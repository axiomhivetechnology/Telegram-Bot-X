# Intermediate Python Bot

This template demonstrates how to handle complex interactions using a **Conversation Handler**.

## Features
- **Custom Keyboards**: Shows buttons to the user.
- **State Management**: Remembers what step of the conversation the user is in.
- **Handling Media**: Can receive and save photos.
- **Handling Location**: Can process GPS coordinates.

## How to Run

1.  **Install requirements**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **Set your Token**: Create a `.env` file with `TELEGRAM_TOKEN`.
3.  **Run the bot**:
    ```bash
    python bot.py
    ```
